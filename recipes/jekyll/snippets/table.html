# Table Read a CSV or YAML or JSON file in `_data` directory and render as a table. See also [Tabulate CSV Data](https://jekyllrb.com/tutorials/csv-to-table/) on the Jekyll docs. {% raw %} - `_data/authors.csv` First name,Last name,Age,Location John,Doe,35,United States Jane,Doe,29,France Jack,Hill,25,Australia - `index.html` ```liquid --- title: Table test --- {% for row in site.data.authors %} {% if forloop.first %} {% for pair in row %} {% endfor %} {% endif %} {% tablerow pair in row %} {{ pair[1] }} {% endtablerow %} {% endfor %}
{{ pair[0] }}
``` {% endraw %}
